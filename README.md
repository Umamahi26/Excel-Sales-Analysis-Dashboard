# Excel-Sales-Analysis-Dashboard
Retail Sales Analysis using Excel to evaluate revenue trends, regional performance, product category contribution, customer insights, and profitability. Includes data cleaning, pivot-based analysis, KPI metrics, and an interactive dashboard with slicers to support data-driven business decisions and strategic planning.
# Sales Performance Analysis using Excel

## Project Objective
To analyze retail sales data and identify revenue trends, regional performance,
customer contribution, and profitability insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard
- Data Cleaning Functions

## Key Insights
- West region generated highest sales
- Technology category produced maximum profit
- Some sub-categories like Tables show consistent losses
- Sales peak during November–December
- Few customers contribute large revenue share

## Dashboard Features
- Interactive slicers
- KPI summary cards
- Monthly trend visualization
- Region & category performance charts

## Business Impact
- Focus marketing in high-performing regions
- Reduce discounts on loss-making products
- Improve retention strategy for repeat customers
- Optimize inventory based on monthly demand trends
